# scripts
## collect-status.sh

Install:

```bash
mkdir -p ~/.local/bin
cp collect-status.sh ~/.local/bin/collect-status.sh
chmod +x ~/.local/bin/collect-status.sh
```

Examples:

```bash
collect-status.sh --brief
collect-status.sh --full
collect-status.sh --brief --network --ollama
collect-status.sh --ollama
```
